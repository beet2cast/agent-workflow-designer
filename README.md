# 🧩 Agent 逻辑编排器

> 给新手用的 AI Agent 协作逻辑搭建工具  
> 可视化排列你的智能体们谁先做、谁后做、怎么做、做完了给谁

**用浏览器打开 `index.html` 就能用，零安装零依赖，一分钟上手。**

---

## 这玩意儿是干什么的？

你有没有想过："我有好几个 AI 智能体要做一件事，怎么安排它们配合？"

这个工具就是干这个的——**用搭积木的方式，画清楚你的 Agent 们怎么协作**。

- 拖一个「智能体」节点到画布上，连上另一个节点
- 谁先做、谁后做、做完交给谁、条件判断走哪条路——全用连线画出来
- 画完了点「运行」，看它怎么一步步执行

**一句话**：把脑子里的 Agent 协作逻辑，用图形画出来、跑一遍、看效果。

---

## 能搭什么样的逻辑？

- **串行**：A → B → C（最简单的流水线）
- **条件分支**：判断结果 → 走真路 / 走假路
- **并行**：同时分发给多个 Agent 处理
- **循环/反馈**：质检不过关 → 返回修改 → 再次质检

内置 **4 个预设模板**，点一下就能看效果。

---

## 5 种节点

| 节点 | 干什么用 |
|------|---------|
| ▶ 开始 | 表示整个流程的起点 |
| ⚡ 智能体 | 让一个 AI 做一件事（分析、写作、判断...） |
| ◆ 条件 | 根据某个条件走不同的分支 |
| ⇄ 并行 | 同时分发给多个智能体 |
| ⏹ 结束 | 流程收尾 |

---

## ✨ 功能一览

- **拖拽搭建** — 节点拉到画布，连线编排
- **画布操作** — 滚轮缩放、拖拽平移
- **条件分支** — 连线带标签（"真"/"假"），条件判断后走对应路线
- **模拟运行** — 跑一遍流程，节点高亮显示执行状态
- **步进调试** — 设断点，点一下走一个节点
- **变量面板** — 实时看上下文数据怎么变
- **深色/浅色** — 右上角一键切换
- **6 家 AI 厂商** — OpenAI / DeepSeek / 阿里百炼 / 豆包 / 通义千问 预设
- **AI 自动填充** — 选中节点，点一下 AI 帮你写处理逻辑
- **保存/加载** — 导出 JSON 文件，下次再打开
- **导出图片** — 把画好的逻辑图存为 PNG
- **撤销/重做** — Ctrl+Z / Ctrl+Shift+Z
- **验证检查** — 自动看出孤立节点、循环风险

## 快速上手

1. 浏览器打开 `index.html`
2. 左侧拖个「开始」到画布
3. 再拖个「智能体」到右边
4. 从开始节点的右侧 **●** 拖到智能体节点的左侧 **●** 连线
5. 继续加节点、连线，搭出你要的流程
6. 点「运行」看效果

> 想配 AI 模型？点右上角「API 设置」，选厂商填 Key 就行。
