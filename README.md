# 🧩 Agent 工作流设计器

> 拖拽式 AI Agent 协作流程可视化编排工具

**在线体验**：打开 `index.html` 即可使用，零安装，零依赖。

---

## ✨ 特性

- **拖拽搭建** — 从左侧面板拖拽节点到画布，连线编排协作流程
- **5 种节点** — 开始、智能体、条件、并行、结束
- **可视化连线** — 从输出端口（右侧 ●）拖到输入端口（左侧 ●）建立连接
- **画布操作** — 滚轮缩放、拖拽平移
- **深色/浅色** — 双主题一键切换
- **条件分支** — 根据连线标签（"真"/"假"）路由到不同路径
- **模拟运行** — 运行工作流并实时查看节点执行状态
- **步进调试** — 设断点，单步执行每个节点
- **变量面板** — 实时查看工作流上下文变量变化
- **6 家模型** — 内置 OpenAI、DeepSeek、阿里百炼、豆包/火山、通义千问预设
- **AI 生成** — 自动为节点生成处理逻辑描述
- **预设模板** — 简单线性 / 质检循环 / 并行处理 / 多路分支
- **保存/加载** — 导出 JSON 工作流文件，随时恢复
- **导出图片** — 将工作流导出为 PNG 图片
- **撤销/重做** — Ctrl+Z / Ctrl+Shift+Z
- **验证器** — 自动检查工作流中孤立节点、循环、配置缺失等问题

---

## 🚀 快速开始

1. 用浏览器打开 `index.html`
2. 从左侧面板拖拽节点到画布
3. 从节点的右侧 ● 拖到下一个节点的左侧 ● 连线
4. 点击配置右侧 AI 模型（可选）
5. 点击「运行」查看效果

🔑 **API Key 配置**：点击右上角「API 设置」，选择厂商并填入 Key 即可使用 AI 生成功能。
